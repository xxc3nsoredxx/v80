# v80

# Registers

## Main (alt) Registers

| Accumulator | Width (bits) | Flag | Width (bits) |
| ----------- | ------------ | ---- | ------------ |
| A(')        | 8            | F(') | 8            |
| B(')        | 8            | C(') | 8            |
| D(')        | 8            | E(') | 8            |
| H(')        | 8            | L(') | 8            |

## Special Registers

| Name | Purpose          | Width (bits) |
| ---- | ---------------- | ------------ |
| I    | Interrupt Vector | 8            |
| R    | Memory Refresh   | 8            |
| IX   | Index Register   | 16           |
| IY   | Index Register   | 16           |
| SP   | Stack Pointer    | 16           |
| PC   | Program Counter  | 16           |
